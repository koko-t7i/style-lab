# Product Type → Recommended Style Sets

These are starting points for picking a 3–5 style set when you know the product type. They are **not prescriptive** — adapt based on the actual brand voice. The point is to give you a fast first draft so you don't waste a turn picking from scratch.

Each set is intentionally varied: one safe, one trendy, one bold, optionally one wildcard. The user needs contrast to articulate preference.

---

## Developer SaaS / DevTools / API products
- Modern Dark (OLED) — the safe one, devs expect dark mode
- Bento Grid — the trendy one, signals "modern AI-era product"
- Brutalism / Neubrutalism — the bold one, signals "indie & opinionated"
- Terminal / E-Ink — the wildcard, signals "hacker / no-bullshit"

## Enterprise / B2B SaaS
- Minimalism / Swiss Modernism — the safe one, signals "credible"
- Bento Grid — the modern one, "we're not your grandfather's enterprise"
- Editorial Grid — the premium one, "we belong in The Economist"
- (skip the loud options — enterprise buyers reject them)

## Consumer AI product (chat, generation, agents)
- Aurora UI / Gradient Mesh — the trendy default for AI in 2025-26
- Modern Dark — the serious option
- Glassmorphism — the dreamy option
- Bento Grid — the structured option

## Fintech / Payments / Banking
- Minimalism — credibility, the floor
- Modern Dark — premium, "private banking" feel
- Editorial Grid — trust + sophistication
- (avoid Brutalism, Cyberpunk — they erode trust)

## Crypto / Web3 / DeFi
- Cyberpunk / HUD — the on-brand option
- Modern Dark — the legit-finance option (signals you're not a scam)
- Brutalism — the degen option
- Aurora UI — the polished-for-normies option

## E-commerce (general)
- Editorial Grid / Magazine — premium products
- Minimalism — clean conversion-focused
- Bento Grid — modern lifestyle brand
- Y2K or Memphis — Gen-Z / fashion / streetwear

## E-commerce (luxury)
- Editorial Grid — the only correct default
- Minimalism — the gallery option
- Modern Dark — evening-wear / jewelry
- (no loud styles — luxury demands restraint)

## Marketing site / Agency / Portfolio
- Brutalism — the "we have a point of view" option
- Editorial Grid — the "we're a real studio" option
- Aurora UI / Glassmorphism — the "we do digital" option
- Exaggerated Minimalism — the "less is more" option

## Productivity / Notes / Writing tool
- Minimalism — get out of the way
- Terminal / E-Ink — for the hacker writers
- Modern Dark — for night writers
- Bento Grid — feature-rich showcase

## Healthcare / Wellness / Mental Health
- Aurora UI / Gradient Mesh — calming, default
- Organic / Biophilic — nature-aligned
- Minimalism — clinical / clean
- Claymorphism — friendly / approachable
- (avoid Cyberpunk, Brutalism, Y2K — wrong tonal registers)

## Gaming / Entertainment
- Cyberpunk / HUD — gamer audience
- Y2K / Memphis — casual / mobile gaming
- Modern Dark — premium / AAA
- Brutalism — indie

## Education / EdTech / Courses
- Editorial Grid — serious / academic
- Bento Grid — modern / feature-showcase
- Claymorphism — younger audiences
- Minimalism — universal default

## Mobile-first apps (general)
- Modern Dark — premium feel
- Material You — Android-native vibe
- Bento Grid — feature-showcase
- Bold Typography — statement / lifestyle

## When the product type isn't here

Map it to the closest one, or compose your own set using these rules:
1. **One floor option** the user can default to (usually Minimalism or Modern Dark depending on brightness preference)
2. **One trendy option** that signals "we know what current good design looks like" (Bento, Aurora, Glassmorphism in 2025-26)
3. **One opinionated option** that takes a real stance (Brutalism, Editorial, Cyberpunk depending on audience)
4. **Optionally one wildcard** that probably won't win but reveals the user's actual preferences via their reaction
